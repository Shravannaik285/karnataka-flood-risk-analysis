🌧 Karnataka Flood Risk Analysis

A data analysis project that examines district-wise rainfall data of Karnataka using Python and Pandas to identify high-risk and low-risk districts based on annual rainfall.

🛠 Technologies Used

- Python 3.x
- Pandas
- NumPy
- Jupyter Notebook
- Git & GitHub

---

🚀 Features

- Import and analyze real-world rainfall dataset (.csv)
- Cleaned column names and handled formatting issues
- Performed filtering and sorting operations
- Identified high rainfall (dangerous) districts
- Identified low rainfall (safe) districts
- Used `.head()` and `.tail()` to display top results
- Compared Annual Rainfall values for classification

⌨ Keyboard Shortcuts (Jupyter Notebook)

- `Shift + Enter` → Run current cell
- `A` → Add cell above
- `B` → Add cell below
- `M` → Convert to Markdown
- `Y` → Convert to Code
- `Ctrl + S` → Save notebook

🔎 Process Followed

1. Loaded dataset using `pd.read_csv()`
2. Cleaned column names using `.str.strip()`
3. Inspected data using:
   - `df.head()`
   - `df.shape`
   - `df.columns`
4. Converted required columns to numeric format
5. Applied filtering conditions using:
   - `>`
   - `<`
   - `&`
6. Sorted districts by rainfall using `.sort_values()`
7. Displayed top dangerous and safest districts
